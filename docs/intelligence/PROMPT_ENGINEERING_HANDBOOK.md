## 1. The G5 Prompting Standard: "Identity-First"

Every G5 agent is primed using a multi-layered identity structure. This ensures the model adopts a highly specific cognitive persona rather than a generic assistant.

### **Core Components of a G5 Prompt:**

1.  **IDENTITY**: The Name and Designation (e.g., SN-00 Neural Orchestrator).
2.  **ROLE**: The specific function (e.g., Gatekeeper, Forge, Architect).
3.  **KNOWLEDGE BASE**: The mental frameworks and citations to use (e.g., McKinsey 7S, Boyd's OODA Loop).
4.  **SWARM_SYNCHRONIZATION**: Instructions on how to interact with the SwarmBus and which entities to consume/produce.
